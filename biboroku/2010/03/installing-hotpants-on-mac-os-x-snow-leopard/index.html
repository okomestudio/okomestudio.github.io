Installing HOTPANTS on Mac OS X Snow Leopard

Posted on March 18, 2010 by Taro

You can’t have enough of hot pants in your life, so you need to reinstall HOTPANTS on Snow Leopard.

First of all, you need a development version of CFITSIO already installed.  See this page if you need to install it yourself.

The version here is 5.1.10.  Download it from Andrew Becker’s distribution web page.

$ mkdir /usr/local/src/hotpants
$ cd /usr/local/src/hotpants
$ wget http://www.astro.washington.edu/users/becker/software/hotpants_v5.1.10.tar.gz
$ tar xf hotpants_v5.1.10.tar.gz
$ cd hotpants_v5.1.10

Now you need to patch the source files for OS X.  The patch that I made is here, which you need to rename it (it is a simple text file, not a real .odt file; it’s a workaround to bypass a wordpress.com limitation):

$ wget ./wp-content/uploads/2010/03/patch_v5-1-10.odt
$ mv patch_v5-1-10.odt patch_v5.1.10
$ patch -p1 < patch_v5.1.10

You need to modify the paths for CFITSIOINCDIR and LIBDIR in Makefile depending on where your CFITSIO is located.  Then do:

$ make
$ sudo cp extractkern hotpants maskim /usr/local/bin

The last line is only necessary if you wish to install the binaries for system-wide access.

That’s it!
